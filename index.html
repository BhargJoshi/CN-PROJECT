SIP Protocol
Session Initiation Protocol handles call setup, teardown, ringing, and codec negotiation. Port 5060 (UDP/TCP).
RTP Protocol
Real-time Transport Protocol carries the actual audio stream. Uses dynamic UDP ports (16384–32767). 50 packets/sec (G.711).
Codecs
G.711 (64 Kbps uncompressed), G.729 (8 Kbps compressed), Opus (6–510 Kbps adaptive). Negotiated via SDP in INVITE.
